Implementação adicionada:
- index.html agora chama POST /api/placafipe
- api/placafipe.js criado para uso no Vercel

Configuração necessária no Vercel:
1. Project Settings > Environment Variables
2. Criar variável: PLACAFIPE_API_KEY
3. Valor: sua chave real da API PlacaFipe
4. Fazer novo deploy

Fluxo:
- No cadastro de veículo: botão "Consultar por Placa"
- Na tela do veículo: botão "Consultar por Placa"
- O app preenche automaticamente placa, marca, modelo, ano, cor, combustível e FIPE quando a API devolver esses dados.

Observação:
- A chave da API não fica no frontend.
- A rota /api/placafipe é executada no backend do Vercel.
